## Run Locally

Clone the project

```bash
  git clone https://github.com/MaxBenDuah/Cookepedia-React.git
```

Navigate to the project directory

```bash
  cd Cookepedia-React
```

Run the project in your browser

```bash
  npm run dev
```

Open the project in your browser
